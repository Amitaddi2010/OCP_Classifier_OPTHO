About This Project

Pilot study for OCP classification using deep learning

🔬 Research Context

This is a pilot study investigating deep learning models for binary classification of ophthalmology images, specifically differentiating Ocular Cicatricial Pemphigoid (OCP) from Normal Eyes.

🤖 Explainable AI

Every prediction includes a Grad-CAM heatmap that visualizes which regions of the eye image the model focused on, providing transparency and clinical interpretability.

📊 Ensemble Approach

The "Compare All" mode aggregates predictions from all 4 models to produce a robust ensemble prediction, reducing individual model bias and improving reliability.
